Windows

Windows 10

变量名 变量值
JAVA_HOME C:\Program Files\Java\jdk1.8.0_131【修改为自己的JDK安装地址】
Path %JAVA_HOME%\bin
%JAVA_HOME%\jre\bin
CLASSPATH .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

Windows 7

变量名 变量值
JAVA_HOME C:\Program Files\Java\jdk1.8.0_131【修改为自己的JDK安装地址】
Path ;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
CLASSPATH .;%JAVA_HOME%\lib;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;

注：两个表格内容一致，第二个表格未进行分行，方便根据系统情况复制。
